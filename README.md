

## Description

This application use to detect fraud depending upon transaction id and confidence level.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Documentation

```bash

# Auto created documentation
$ npm run doc

Documentation file is not hosted. Run this command locally and open index.html file from documentation folder.

```
